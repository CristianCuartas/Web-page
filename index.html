Sobre mi

Mi nombre es Cristian y tengo 19 años, soy chico joven y apasionado por colocar en marcha mis ideas, habitualmente me enfoco en la constante actualización de conocimientos además de mantener conversaciones con otras personas que me ayuden a mejorar habilidades en mi crecimiento profesional.

Me dedico actualmente al desarrollo de software como programador web, empleando las últimas tecnologías JavaScript con uno de sus frameworks React JS, además del manejo de Wordpress, HTML y CSS3. En mi entorno uso manejadores de paquetes y repositorios en la nube GitHub, además de hacer deployment a servidores online para qué pueda visualizar el trabajo que se desarrolla. Si estas buscando hacer una página web con las últimas tecnologías realmente brillante, funcional y a tu gusto entonces llegaste al lugar correcto.

Podrás ponerte en contacto conmigo a través de mi correo electrónico o completando el formulario de contacto aquí mismo en mi página web, yo me encargo del resto.

Servicios

Diseño Web

Páginas web que resalten lo que deseas inspirar, desarrollaré tu idea. La importancia del diseño web es fundamental para tener éxito y atraer a las personas creándole una experencia dentro dentro del sitio.

Responsive desing

El diseño web adaptable, es una filosofía de diseño y desarrollo cuyo objetivo es adaptar la apariencia de las páginas web al dispositivo que se esté utilizando para visitarlas.

SEO

El posicionamiento en buscadores, optimización en motores de búsqueda es un conjunto de acciones orientadas a mejorar el posicionamiento de un sitio web en la lista de resultados de Google, Bing, u otros buscadores de internet.​

Gestión de paquetes

Herramientas esenciales de desarrollo de JavaScript, npm es la herramienta utilizada por más de 11,000,000 desarrolladores de JavaScript en todo el mundo. Sus desarrolladores ya lo usan.

Platform as a Service

Heroku es uno de los PaaS más utilizados en la actualidad en entornos empresariales por su fuerte enfoque en resolver el despliegue de una aplicación. Ademas te permite manejar los servidores y sus configuraciones, escalamiento y la administración. A Heroku solo le dices qué lenguaje de backend estás utilizando o qué base de datos vas a utilizar y te preocupas únicamente por el desarrollo de tu aplicación.

React JS

React es una biblioteca Javascript para crear interfaces de usuario enfocada en la visualización.

¿Por qué trabajar conmigo?

  • Velocidad
  • Me aseguro de que tu web cargue entre 2 y 4 segundos (depende la cantidad de contenido del sitio). Si una web tarda más de 5 segundos en cargar, el 70% de los usuarios lo cierra y no vuelve a entrar más.
  • Asesoramiento
  • Mantengo una comunicación constante con mi cliente, para enseñarle a modificar contenido mediante el panel de Wordpress o para cualquier duda / cambios que quiera hacer en su sitio web..
  • Mobile Priority
  • Cada vez más personas navegan desde sus celulares en cualquier parte del mundo. Me aseguro de que tu web esté optimizado para cualquier tipo de dispositivo móvil.
